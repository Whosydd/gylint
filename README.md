## 说明

<br>

如果你和我一样懒，请使用这个包。。。

<br>

### Installation

```bash
npm i -D gylint
```

<br>

### Usage

```bash
npx gylint
```

> 在项目根目录使用该命令会将”.eslintrc”、“.prettierrc”以及“.stylelintrc”三个配置文件复制到根目录，如果已存在相关配置文件，则会重命名已存在文件，请知悉~
