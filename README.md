## 说明

<br>

如果你和我一样懒，请使用这个包。。。

<br>

### Installation

```bash
npm i -D gylint
```

<br>

### Usage

```bash
npx gylint
```

> 该命令会在项目根目录创建 `.eslintrc.js` `.eslintignore` `.prettierrc.js` `.prettierignore` `.stylelintrc.js` `.stylelintignore`
>
> 你可以根据你的需要去修改它
